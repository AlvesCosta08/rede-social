# Social Network Frontend

## Descrição

Este é o frontend de uma rede social desenvolvida com **React**. O projeto é parte de uma aplicação de rede social completa que utiliza **Go** para o backend. O frontend é responsável por fornecer a interface de usuário interativa e responsiva, permitindo aos usuários criar e gerenciar perfis, conectar-se com outros usuários e interagir com postagens e comentários.

## Funcionalidades

- **Autenticação e Registro**: Permite que os usuários se registrem, façam login e gerenciem suas sessões.
- **Perfil do Usuário**: Exibe informações do perfil, permite a atualização de dados e a visualização de postagens e conexões.
- **Feed de Notícias**: Mostra atualizações e postagens dos amigos e seguidores.
- **Postagens**: Criação, edição e exclusão de postagens, com suporte para imagens e texto.
- **Comentários e Curtidas**: Interação com postagens através de comentários e curtidas.
- **Busca e Conexões**: Pesquisa de usuários e adição de conexões/amigos.

## Tecnologias Utilizadas

- **React**: Biblioteca JavaScript para construir interfaces de usuário.
- **React Router**: Gerenciamento de navegação e roteamento.
- **Redux**: Gerenciamento de estado global.
- **Axios**: Cliente HTTP para comunicação com a API backend.
- **CSS Modules**: Escopo de estilo modular e isolado.
